== Description ==

老部落百度提交插件，包括百度普通提交和百度快速提交，提高百度爬取网站和加快百度收录网站。插件是基于百度资源平台快速提交和普通提交API，我们只是做到一个简单的自动工具。公众号：老蒋朋友圈。

## 插件特点

1. 可选普通和快速提交，完全免费
2. 可批量普通和快速提交
3. 自动检查网站文章收录情况

插件安装方法：[https://www.lezaiyun.com/baidu-submit.html](https://www.lezaiyun.com/baidu-submit.html)

== Installation ==

* 把laobuluo-baidu-submit文件夹上传到/wp-content/plugins/目录下<br />
* 在后台插件列表中激活laobuluo-baidu-submit<br />
* 在WP左侧菜单看到百度提交插件<br />
* 填写我们在百度资源平台的TOKEN

== Frequently Asked Questions ==

* 当发现插件出错时，开启调试获取错误信息。
* 根据需要我们选择是否开启收录检索功能
* 根据需要我们选择开启自动普通提交（编辑页面看到默认勾选）
* 如果快速额度不足，提交快速收录是无用的
